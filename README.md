# Fusión Gourmet Festival – Landing Page

## Description

A landing page built for **Fusión Gourmet**, a fictional avant-garde gastronomic festival that celebrates creative cuisine and innovative culinary experiences. The design communicates elegance, flavor, and modernity to attract food lovers and encourage them to book tickets.

## Technologies Used

- **HTML5** – Semantic structure (`header`, `main`, `section`, `article`, `nav`, `footer`)
- **CSS3** – Custom properties, Flexbox, Grid Layout, Media Queries, animations
- **JavaScript** – Minimal vanilla JS for the mobile hamburger menu toggle

## Project Structure

```
fusion-gourmet/
├── index.html    ← Main HTML file with all sections
├── style.css     ← All styles (colors, layout, responsive)
└── README.md     ← This file
```

## Sections Included

| Section | Description |
|---|---|
| Header | Fixed navigation bar with logo and nav links |
| Hero | Full-screen introduction with call to action |
| Schedule | HTML table with tasting and workshop timetable |
| Chefs | Grid of featured guest chefs |
| About | Festival philosophy and sponsors (with `<ul>` lists) |
| Gallery | CSS Grid image gallery + video placeholder |
| Footer | Contact info and social media links |

## How to View the Project Locally

1. Download or clone the project folder.
2. Open the `index.html` file directly in any modern browser (Chrome, Firefox, Edge).
3. No server or installation required — it runs entirely in the browser.

## Responsive Breakpoints

- **Desktop** (`> 1024px`): Full layout with multi-column grids
- **Tablet** (`≤ 900px`): Adjusted grid columns
- **Mobile** (`≤ 640px`): Single column, hamburger menu activated
